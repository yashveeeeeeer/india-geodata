Download by format

# Download only Parquet files from a release
gh release download admin/states --pattern "*.parquet"

# Download only PMTiles files
gh release download admin/states --pattern "*.pmtiles"

# Download only GeoJSON files
gh release download remote-sensing/nightlights --pattern "*.geojson"

Bulk download

# Download all release assets (~35 GB)
python scripts/download-releases.py

# Or use gh CLI to list all releases
gh release list

In-repo data

Smaller datasets (shapefiles, GeoJSON, KML, CSV) are stored directly in the data/ directory. Clone the repo or download individual files from the catalog.

External datasets

The SHRUG platform by Development Data Lab provides additional socioeconomic and geospatial data for India at the village and town level. Visit their download portal directly.
