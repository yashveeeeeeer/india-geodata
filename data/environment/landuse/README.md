# Land Use Classification Data

Land use classification boundary data. Currently covers Hyderabad only.

## In-repo Files

Located in the `hyderabad/` subdirectory:

| File | Format | Description |
|------|--------|-------------|
| `Hyderabad_Landuse.geojson` | GeoJSON | Hyderabad land use classification polygons |
| `Hyderabad_Landuse.zip` | Shapefile (zipped) | Hyderabad land use classification polygons |
