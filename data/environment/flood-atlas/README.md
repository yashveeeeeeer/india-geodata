# India Flood Atlas

District-wise, state-wise, and sub-basin-wise flood statistics for India
(1901–2020), produced by the Water and Climate Lab at IIT Gandhinagar.

## Contents

| File | Description | Size |
|---|---|---|
| `District_Wise_annual_flood_area.json` | Annual flooded area per district (1901–2020) | 1.6 MB |
| `District_Wise_annual_flood_frac.json` | Annual flood fraction per district | 1.5 MB |
| `District_Wise_flood_risk_data.json` | District-level flood risk metrics | 90 KB |
| `District_Wise_max_flood_area_frac.json` | Maximum flood area fraction per district | 49 KB |
| `State_Wise_annual_flood_area.json` | Annual flooded area per state | 81 KB |
| `State_Wise_annual_flood_frac.json` | Annual flood fraction per state | 72 KB |
| `State_Wise_flood_risk_data.json` | State-level flood risk metrics | 5 KB |
| `State_Wise_max_flood_area_frac.json` | Maximum flood area fraction per state | 2 KB |
| `Subbasin_Wise_flood_risk_data.json` | Sub-basin flood risk metrics | 9 KB |

## Methodology

Flood areas were reconstructed using the combined H08-CaMa-Flood hydrological
and hydrodynamic model at 10 km spatial resolution. The role of 51 major
reservoirs was considered in the flood risk assessment.
