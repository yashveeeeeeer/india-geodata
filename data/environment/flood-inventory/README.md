# India Flood Inventory (IFI v3.0)

Multi-source geospatial inventory of flood events across India, produced by
the HydroSense Lab at IIT Delhi in collaboration with IMD.

## Contents

| File | Description | Size |
|---|---|---|
| `INDIA_FLOOD_INVENTORY_V3.geojson` | 1,006 flood event polygons with dates, states, districts, and causal factors | ~28.6 MB |

## Attributes

- **StartDate / EndDate** — Flood event date range
- **State** — Affected state(s)
- **Districts** — Affected district(s)
- **MainCause** — Primary cause (monsoon, cyclone, dam release, etc.)
- **EventSour** — Source of event report
