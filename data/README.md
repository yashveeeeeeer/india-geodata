# Data Directory

Root directory for all geodata. Browse subdirectories by category:

- [administrative/](administrative/) -- Country, state, and district boundaries
- [electoral/](electoral/) -- Assembly and parliamentary constituency boundaries
- [census/](census/) -- Census-derived datasets
- [environment/](environment/) -- Environmental and ecological data
- [infrastructure/](infrastructure/) -- Roads, transport, and rural infrastructure
- [urban/](urban/) -- Urban area data
- [postal/](postal/) -- Pincode boundaries
- [police/](police/) -- Police jurisdiction boundaries
- [survey-of-india/](survey-of-india/) -- Survey of India reference maps and indices

See the [repository README](../README.md) for full documentation.
