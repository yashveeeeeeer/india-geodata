# Airports & Aerodromes

Point and polygon geometries of airports, aerodromes, helipads, and emergency landing sites across India, extracted from OpenStreetMap via the Humanitarian OpenStreetMap Team (HOT) export service.

## Data

| File | Format | Size | Description |
|------|--------|------|-------------|
| `INDIA_AIRPORTS_POINTS.geojson` | GeoJSON | 1.1 MB | Point locations of airports, aerodromes, helipads, and landing sites |
| `INDIA_AIRPORTS_POLYGONS.geojson` | GeoJSON | 5 MB | Polygon outlines of airport and aerodrome areas |

Both files are stored in this directory.

## Attributes

Point and polygon features with properties including facility name, type (airport, aerodrome, helipad), operator, and capacity where available. Attribute coverage depends on OpenStreetMap contributor input.

## Sources

- **Humanitarian OpenStreetMap Team (HOT)** — https://data.humdata.org/dataset/hotosm_ind_airports
