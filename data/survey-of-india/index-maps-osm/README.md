# Topographic Sheet Index Grids (OSM / UTM / WGS84 System)

Survey of India topographic sheet index grids using the OSM/UTM/WGS84 coordinate system.

**Included scales:**
- `OSM_1M_Index` -- 1:1,000,000
- `OSM_250K_Index` -- 1:250,000
- `OSM_50K_Index` -- 1:50,000

- **Format:** Shapefile (one set per scale)
- **Source:** Survey of India

See the [repository README](../../../README.md) for full documentation.
