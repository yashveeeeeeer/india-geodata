# Survey of India Reference Maps and Index Grids

Reference boundary data and topographic index grids from the Survey of India (SOI).
All files are stored in-repo.

## Directory Structure

| Subdirectory | Description |
|--------------|-------------|
| `india-boundary/` | National boundary of India (Shapefile) |
| `state-boundary/` | State boundaries of India (Shapefile) |
| `index-maps-osm/` | UTM/WGS84 topographic sheet grid indices |
| `index-maps-old-system/` | Everest Polyconic datum grid indices |
| `outline-maps/` | PDF outline maps and vector map shapefile |

## Index Maps (OSM / WGS84)

Topographic sheet index grids on the WGS84 datum at three scales:

| Scale | Description |
|-------|-------------|
| 1:1,000,000 (1M) | Degree sheet grid |
| 1:250,000 (250K) | Quarter-degree sheet grid |
| 1:50,000 (50K) | Topographic sheet grid |

## Index Maps (Old System)

Topographic sheet index grids on the Everest Polyconic datum, corresponding to the
legacy Survey of India numbering system.

## Outline Maps

PDF outline maps of India (political and physical) along with a vector map shapefile.
