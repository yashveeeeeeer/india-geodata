# Topographic Sheet Index Grids (Old Everest Polyconic System)

Survey of India topographic sheet index grids using the old Everest Polyconic coordinate system.

**Included scales:**
- `OldSystem_1M_Index` -- 1:1,000,000
- `OldSystem_250K_Index` -- 1:250,000
- `OldSystem_50K_Index` -- 1:50,000

- **Format:** Shapefile (one set per scale)
- **Source:** Survey of India

See the [repository README](../../../README.md) for full documentation.
