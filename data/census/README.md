# Census Boundary Data

Administrative boundary datasets derived from the Census of India and related sources.

## Sub-datasets

| Directory | Description | Period |
|-----------|-------------|--------|
| `census-2011/` | Administrative boundaries from Census 2011 | 2011 |
| `historical/` | Historical district boundaries | 1941--2001 |

## Sources

- **Census of India** -- Official census administrative boundaries
- **Development Data Lab (SHRUG)** -- Socioeconomic High-resolution Rural-Urban Geographic dataset
