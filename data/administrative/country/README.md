# India Country Boundary

National boundary of India at the country level.

## Files

| File | Description |
|------|-------------|
| `india-composite.geojson` | Composite boundary of India |
| `india-soi.geojson` | Boundary as per Survey of India |
| `india-osm.geojson` | Boundary sourced from OpenStreetMap |

### Disputed Territories

The `disputed/` subfolder contains boundary files for disputed and
claimed territories.

## Source

All files are sourced from the
[DataMeet Maps repository](https://github.com/datameet/maps).

## Coordinate Reference System

WGS 84 (EPSG:4326)
