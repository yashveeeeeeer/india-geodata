# Disputed Territory Boundaries

Boundary files for disputed territories including Pakistan-occupied Kashmir (PoK) and the Shaksgam Tract.

**Files:**
- `india-disputed-lsib.geojson` -- Disputed boundary from LSIB dataset
- `pok-alhasan.geojson` -- PoK boundary from Alhasan dataset
- `pok-districts-alhasan.geojson` -- PoK district-level boundaries from Alhasan dataset
- `shaksgam-ne.geojson` -- Shaksgam Tract boundary

**Sources:** LSIB (Large Scale International Boundaries), Alhasan Systems

See the [repository README](../../../../README.md) for full documentation.
