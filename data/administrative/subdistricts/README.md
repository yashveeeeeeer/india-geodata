# Subdistrict Boundaries

Subdistrict-level (Tehsil / Taluk / Mandal) administrative boundaries of India.

## Files

All files are distributed via GitHub Releases under the tag **`admin/subdistricts`**.

| File | Source | Format |
|------|--------|--------|
| `LGD_Subdistricts.parquet` | Local Government Directory | Parquet |
| `LGD_Subdistricts.pmtiles` | Local Government Directory | PMTiles |
| `LGD_Subdistricts.geojsonl.7z` | Local Government Directory | GeoJSONL (7z) |
| `SOI_Subdistricts.parquet` | Survey of India | Parquet |
| `SOI_Subdistricts.pmtiles` | Survey of India | PMTiles |
| `SOI_Subdistricts.geojsonl.7z` | Survey of India | GeoJSONL (7z) |

**Total**: 6 files

### Download

```bash
gh release download admin/subdistricts -R <owner>/india-geodata
```
