# Block Boundaries

Block-level administrative boundaries of India.

## Files

All files are distributed via GitHub Releases under the tag **`admin/blocks`**.

| File | Source | Format |
|------|--------|--------|
| `LGD_Blocks.parquet` | Local Government Directory | Parquet |
| `LGD_Blocks.pmtiles` | Local Government Directory | PMTiles |
| `LGD_Blocks.geojsonl.7z` | Local Government Directory | GeoJSONL (7z) |
| `PMGSY_Blocks.parquet` | PMGSY (Pradhan Mantri Gram Sadak Yojana) | Parquet |
| `PMGSY_Blocks.pmtiles` | PMGSY (Pradhan Mantri Gram Sadak Yojana) | PMTiles |
| `PMGSY_Blocks.geojsonl.7z` | PMGSY (Pradhan Mantri Gram Sadak Yojana) | GeoJSONL (7z) |
| `bhuvan_blocks.parquet` | Bhuvan (ISRO) | Parquet |
| `bhuvan_blocks.pmtiles` | Bhuvan (ISRO) | PMTiles |
| `bhuvan_blocks.geojsonl.7z` | Bhuvan (ISRO) | GeoJSONL (7z) |

**Total**: 9 files

### Download

```bash
gh release download admin/blocks -R <owner>/india-geodata
```
