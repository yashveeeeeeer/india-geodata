# Municipal Ward Boundaries

Municipal ward boundaries for 28 Indian cities. Data is stored in-repo in city-specific
subdirectories. Typical data per city includes ward boundaries, municipal boundary, zones,
and facilities in GeoJSON, KML, and Shapefile formats.
