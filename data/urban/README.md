# Urban / Municipal Data

Geospatial datasets for urban areas across India, covering municipal boundaries,
ward delineations, slum extents, and locality boundaries.

## Sub-datasets

| Directory | Description | Coverage |
|-----------|-------------|----------|
| `municipal-boundaries/` | Municipal ward and zone boundaries | 28 cities |
| `wards/` | Ward boundary data from national-level sources | Pan-India |
| `slums/` | Slum boundary delineations | Select cities/states |
| `localities/` | Locality boundary data | Select cities |

## Sources

- DataMeet Municipal Spatial Data
- Swachh Bharat Mission (SBM)
- World Bank AMRUT datasets
- State-level GIS portals (TNGIS, TRACGIS, BBMP, DUSIB, etc.)
- Living Atlas of the World
