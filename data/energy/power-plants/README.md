# Power Plants

Point locations of power plants across India.

## Data

| File | Format | Size |
|------|--------|------|
| INDIA_ENERGY_PLANTS.geojson | GeoJSON | ~128 KB |

## Attributes

| Field | Description |
|-------|-------------|
| `plant_name` | Name of the power plant |
| `district` | District where the plant is located |
| `state` | State where the plant is located |
| `inst_cap` | Installed capacity in MW |
| `type` | Fuel type (coal_power_plant, diesel_power_plant, hydro_power_plant) |
