# Police Station Locations

Point locations of police stations across India.

## Data

| File | Format | Size |
|------|--------|------|
| INDIA_POLICE_STATIONS.geojson | GeoJSON | ~5 MB |

## Attributes

Point features with properties including station name, district, and state.
