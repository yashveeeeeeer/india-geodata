# Wetlands & Ramsar Sites

Polygon boundaries of wetlands across India — including Ramsar-designated wetlands, surveyed wetland boundaries from the Parivesh portal, SOI-mapped salt pans, and swamps.

All files are distributed via GitHub Releases.

## Release Tag

`water/wetlands`

## Datasets

| Dataset | Description | Source |
|---------|-------------|--------|
| Bharatmaps_Parivesh_Ramsar_Wetlands | Ramsar-designated wetland boundaries | BharatMaps / Parivesh |
| Bharatmaps_Parivesh_Wetland_Boundaries | Surveyed wetland boundaries | BharatMaps / Parivesh |
| NCOG_SOI_SaltPans | Salt pan polygons from SOI/NCOG | NCOG / Survey of India |
| NCOG_SOI_Swamps | Swamp polygons from SOI/NCOG | NCOG / Survey of India |

Each dataset is available in Parquet, PMTiles, and GeoJSONL.7z formats.

**Total:** ~935 MB
