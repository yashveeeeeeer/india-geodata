# Urban Water Features

Point and line geometries of urban water infrastructure — stormwater drains, ghats, and urban water bodies from the Swachh Bharat Mission and AMRUT programmes.

All files are distributed via GitHub Releases.

## Release Tag

`water/urban-water`

## Datasets

| Dataset | Description | Source |
|---------|-------------|--------|
| SBM_Ghats | Ghat locations | Swachh Bharat Mission |
| SBM_Stormwater_drains | Stormwater drain lines | Swachh Bharat Mission |
| SBM_Water_bodies | Urban water body locations | Swachh Bharat Mission |
| WB_AMRUT_Stormwater_drains | Stormwater drain lines from AMRUT | AMRUT |
| WB_AMRUT_Stormwater_vents | Stormwater vent point locations from AMRUT | AMRUT |

Each dataset is available in Parquet, PMTiles, and GeoJSONL.7z formats.

**Total:** ~50 MB
