# Water Body Census

Point locations of water bodies from India's first national Water Body Census (Jal Dharohar) — covering ponds, lakes, tanks, and reservoirs with attributes for type, ownership, usage, condition, and encroachment status.

All files are distributed via GitHub Releases.

## Release Tag

`water/waterbody-census`

## Datasets

| Dataset | Description | Source |
|---------|-------------|--------|
| JalDharohar_Waterbody_Census | Water body point locations from the national census | Jal Dharohar |

Each dataset is available in Parquet, PMTiles, and GeoJSONL.7z formats.

**Total:** ~492 MB

## Download

gh release download water/waterbody-census --repo yashveeeeeer/india-geodata
