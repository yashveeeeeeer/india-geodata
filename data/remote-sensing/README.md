# Remote Sensing

Datasets derived from satellite remote sensing, including nighttime lights, land cover, and other Earth observation products.

## Datasets

| Dataset | Description | Coverage | Formats | Size |
|---------|-------------|----------|---------|------|
| [nightlights](nightlights/) | District-level VIIRS nighttime lights (2012–2024) | 641 districts | CSV, GeoJSON | ~353 MB |
| [population-density](population-density/) | 1 km gridded population density (WorldPop 2020) | India (national) | GeoTIFF | ~17 MB |
