# Buildings Data

Building footprint polygon datasets for India — ML-detected and survey-derived footprints from multiple sources.

## Sub-datasets

| Directory | Description | Source | License |
|-----------|-------------|--------|---------|
| `urban/` | Urban building footprints across states/UTs (84 assets, ~9 GB) | AMRUT, Bhuvan, GSDL Delhi, VEDAS, NESDR via ramSeraph | CC0 1.0 |
| `google/` | Google Open Buildings India 2023 (47 assets, ~66 GB) | Google Research Open Buildings V3 via ramSeraph | CC-BY-4.0 |
| `microsoft/` | Microsoft Buildings India (11 assets, ~12 GB) | Microsoft GlobalMLBuildingFootprints via ramSeraph | ODbL |

## Formats

All datasets are available as GeoJSONL (.7z), Parquet, and PMTiles. Data is distributed via [ramSeraph/indian_buildings](https://github.com/ramSeraph/indian_buildings) releases — not hosted in this repository.
