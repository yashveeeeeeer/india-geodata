# Urban Buildings India

Building footprint polygons for urban areas across Indian states and UTs. Aggregated from AMRUT/Bhuvan, GSDL Delhi, VEDAS, NESDR, and other sources.

## Contents

| Item | Details |
|------|---------|
| **Assets** | 84 state/UT-level files |
| **Total size** | ~9 GB |
| **Formats** | GeoJSONL (.7z), Parquet, PMTiles |

**External download:** Data is not hosted in this repository. Download from the source release.

## Source

- **Release:** [ramSeraph/indian_buildings — urban](https://github.com/ramSeraph/indian_buildings/releases/tag/urban)
- **Upstream:** AMRUT, Bhuvan, GSDL Delhi, VEDAS, NESDR
