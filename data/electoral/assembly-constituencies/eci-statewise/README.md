# Assembly Constituency Boundaries (ECI, State-wise)

State-wise Assembly Constituency boundaries from the Election Commission of India.
Pre-2014 delimitation. Shapefile format, one set per state.

- **Coverage:** States S01-S28, UTs U05 and U07
- **Format:** Shapefile (one `.shp`/`.dbf`/`.shx` set per state, e.g. `S01_AC.*`)
- **Source:** DataMeet / Election Commission of India

See the [repository README](../../../../README.md) for full documentation.
