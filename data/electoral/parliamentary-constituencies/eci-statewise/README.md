# Parliamentary Constituency Boundaries (ECI, State-wise)

State-wise Parliamentary Constituency boundaries from the Election Commission of India.
Shapefile format, one set per state or union territory.

- **Coverage:** States S01-S28, UTs U01-U07
- **Format:** Shapefile (one `.shp`/`.dbf`/`.shx` set per state/UT, e.g. `S01_PC.*`)
- **Source:** DataMeet / Election Commission of India

See the [repository README](../../../../README.md) for full documentation.
