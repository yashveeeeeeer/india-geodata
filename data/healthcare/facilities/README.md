# Healthcare Facilities

Point locations of public health facilities across India — primary health centres (PHCs), community health centres (CHCs), sub-centres, district hospitals, and other medical facilities.

## Data

### GitHub Release: `healthcare/facilities` (~47 MB)

| File | Format | Size |
|------|--------|------|
| `INDIA_HEALTH_FACILITIES_NIC.geojson` | GeoJSON | 47 MB |

**Download:**

```bash
gh release download healthcare/facilities --repo yashveeeeeer/india-geodata
```

## Attributes

Point features with properties including facility name, type (PHC, CHC, sub-centre, district hospital, etc.), district, and state.
