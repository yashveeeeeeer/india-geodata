# Urban Roads

Urban road networks and bridge/flyover data from AMRUT cities, Bhuvan NUIS, and Telangana CDMA.

All files are distributed via GitHub Releases.

## Release Tag

`infra/urban-roads`

## Datasets

| Dataset | Description | Source |
|---------|-------------|--------|
| AMRUT_Road_CLines | Road center-lines from AMRUT cities | AMRUT / Bhuvan |
| Bhuvan_NUIS_Urban_Roads | Urban road network from Bhuvan NUIS | Bhuvan NUIS |
| TS_CDMA_Roads | Road network for Telangana | CDMA Telangana |
| AMRUT_Bridges_and_Flyovers | Bridges and flyovers in AMRUT cities | AMRUT / Bhuvan |

Each dataset is available in Parquet, PMTiles, and GeoJSONL.7z formats.

**Total:** ~1.3 GB

## Download

```bash
gh release download infra/urban-roads --repo yashveeeeeer/india-geodata
```
