# ML Road Detections (Microsoft + Facebook)

Machine-learning-detected road networks for India from satellite imagery.

All files are distributed via GitHub Releases.

## Release Tag

`infra/ml-roads`

## Datasets

| Dataset | Description | License | Format |
|---------|-------------|---------|--------|
| ms_roads_india | Microsoft Road Detections | ODbL | Parquet, PMTiles, GeoJSONL.7z |
| fb_roads_india | Facebook MapWithAI Roads | MIT | Parquet, PMTiles, GeoJSONL.7z |

**Total:** ~5.5 GB

## Download

```bash
gh release download infra/ml-roads --repo yashveeeeeer/india-geodata
```
