# PMGSY Road Proposals

Proposed rural road connectivity projects under the PMGSY (Pradhan Mantri Gram Sadak Yojana) programme.
One ZIP file per state, each containing shapefiles.

- **Coverage:** 29 states
- **Format:** State-wise ZIP archives containing shapefiles
- **Source:** GeoSadak / PMGSY

See the [repository README](../../../../README.md) for full documentation.
