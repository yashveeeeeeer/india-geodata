# PMGSY Road Network (DRRP)

Existing and planned rural roads from the District Rural Road Plan (DRRP) under PMGSY.
State-wise ZIP archives containing shapefiles.

Files under 50 MB are stored in this directory. Larger files for Maharashtra, Bihar,
Karnataka, Uttarakhand, Rajasthan, and West Bengal are available via the Git release
tag `infra/rural-roads-large`.

- **Format:** State-wise ZIP archives containing shapefiles
- **Source:** GeoSadak / PMGSY

See the [repository README](../../../../README.md) for full documentation.
