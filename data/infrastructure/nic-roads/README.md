# NIC Roads (Bharatmaps)

Road network from the National Informatics Centre (NIC) Bharatmaps platform.

All files are distributed via GitHub Releases.

## Release Tag

`infra/nic-roads`

## Datasets

| Dataset | Description | Format |
|---------|-------------|--------|
| NIC_Roads | Road network polylines (split Parquet) | Parquet, PMTiles, GeoJSONL.7z |

**Total:** ~5.4 GB

## Download

```bash
gh release download infra/nic-roads --repo yashveeeeeer/india-geodata
```
