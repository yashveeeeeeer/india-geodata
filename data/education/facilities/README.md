# Education Facilities

Point and polygon locations of schools, colleges, universities, and kindergartens across India, extracted from OpenStreetMap via the Humanitarian OpenStreetMap Team (HOT) export service.

## Data

### In this directory

| File | Format | Size | Description |
|------|--------|------|-------------|
| `INDIA_EDUCATION_FACILITIES_POINTS.geojson` | GeoJSON | 7.4 MB | Point locations of education facilities |

### GitHub Release: `education/facilities` (~41 MB)

| File | Format | Size | Description |
|------|--------|------|-------------|
| `hotosm_ind_education_facilities_polygons_geojson.geojson` | GeoJSON | 41 MB | Polygon outlines of education facility areas |

**Download:**

```bash
gh release download education/facilities --repo yashveeeeeer/india-geodata
```

## Attributes

Point and polygon features with properties including facility name, type (school, college, university, kindergarten), operator, capacity, and address where available. Attribute coverage depends on OpenStreetMap contributor input.
